1. Language Modeling Basics

1.1 What is a Language Model?

A language model is a type of machine learning model that predicts the next word in a sequence. It's like predictive text on your phone — given some text, it tries to guess what comes next. Language models are foundational to systems like ChatGPT, autocomplete, translation engines, and more.

Analogy

Think of a language model like a super-smart guesser playing Mad Libs. If you say "The cat sat on the ___", it has seen so many sentences that it can confidently guess "mat" — not because it understands, but because it's statistically the most likely word.

Watch

Example Code

# Basic n-gram-like example in Python
text = "the cat sat on the mat"
tokens = text.split()

def predict_next(tokens, prev_word):
    for i in range(len(tokens)-1):
        if tokens[i] == prev_word:
            return tokens[i+1]
    return None

print(predict_next(tokens, "the"))  # Output: cat or mat

1.2 Bag of Words

How do we teach machines to understand text? One simple but powerful idea is the Bag of Words method. It turns text into numbers so a machine can use it.

Analogy

Imagine each sentence is a shopping basket, and the words are the items inside. A Bag of Words is like counting how many times each item appears. It doesn’t care about order — only what’s in the basket.

Here’s a super simple example: 

# Sample text documents
docs = [
  "I love movies",
  "Movies are fun",
  "I love fun"
]

from sklearn.feature_extraction.text import CountVectorizer
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(docs)

print(vectorizer.get_feature_names_out())
print(X.toarray())

This code turns three small documents into a table of word counts. The result is a simple grid — rows for documents, columns for words, and numbers that show how often each word appears.
